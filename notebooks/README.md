# notebooks/

Jupyter notebooks for exploratory data analysis (EDA) and initial visualizations.

Use this folder for quick, iterative exploration of the survey data — checking distributions,
testing hypotheses, and prototyping charts before turning stable logic into reusable modules
in `src/`.

Naming convention: `NN-short-description.ipynb` (e.g. `01-eda-2025-survey.ipynb`).
